O que ECMAScript tem a ver com JavaScript?

O padrão JavaScript é ECMAScript. Desde 2012, todos os navegadores modernos possuem suporte total ao ECMAScript 5.1. Assim, em poucas palavras, podemos dizer que ECMAScript é uma especificação de linguagem, ou seja, ela define os padrões para uma linguagem de programação, e o JavaScript é a implementação desses padrões.

Java e JavaScript possuem alguma relação?

Não se deve confundir o JavaScript com a linguagem de programação Java. Tanto Java quanto JavaScript são marcas registradas da Oracle nos Estados Unidos da América e em outros países. No entanto, as duas linguagens de programação possuem sintaxe, semânticas e usos muito diferentes.

Para que serve o Node.js ?

Node.js é um ambiente de tempo de execução JavaScript de plataforma cruzada que permite aos desenvolvedores criar aplicativos de servidor e de rede com JavaScript.

Como fazer com que um número digitado em um prompt() possa ser usado em cálculos?

Criando uma variável e atribuindo a ela o prompt utilizando o Number para transformar a string em number inteiro ou real, fazendo da seguinte maneira
var n1 = Number(window.prompt("digite um número"))
var n2 = Number(window.prompt("digite outro número"))
var s = n1 + n2
window.alert(`A soma entre os numeros ${n1} e ${n2} é igual a ${s}`)

Como mostrar um número formatado como um valor monetário?

Utilizando o metodo .toFixed(2) se eu quiser duas casas decimais e se caso eu queira trocar algum objeto, tipo um ponto por uma virgula 1545.50 para 1545,50 eu faço da seguinte maneira .toFixed(2).replace('.',',') nesse comando eu falei que quero trocar o ponto pela vírgula, caso eu queira mostrar esse valor de 1545,50 em um valor monetário eu irei utilizar o metodo .toLocaleString('pt-BR' {style: 'currency', currency : 'BRL'}).replace('.',',') eu quis dizer na seguinte ordem toLocaleString('pt-BR') eu quero localizar para o Brasil e o estilo vai currency que quer dizer dinheiro, e o dinheiro vai ser BRL Brasil Real. Caso eu queira mudar para dolar ou outra moeda é so colocar a sígla da moeda desejada, USD e EUR por exemplo.

Para que serve o operador % em JS? seria para calcular porcentagem?

Não, ele é para calcular uma divisão inteira, exemplo 5 % 2 = 1, resumindo % é o resto de uma divisão

6 + 4 / 2 é igual a 5 ou 8?

Por ordem de precedência o programa irá exucutar primeiro a divisão e depois a soma, no caso o resultado é 8, para mudar a ordem de precedência, podemos fazer da seguinte maneira: (6 + 4) / 2 logo o resultado será igual a 5 pois obrigamos o programa somar primeiro os valores dentro do parentes.

Para que servem os operadores ++ e --

É um incremento como se fosse variavel + 1 ou variavel - 1 exemplo:
var x = 10
x++
x = 11
x--
x = 10

Você já conhece os operadores relacionais e os operadores lógicos em JavaScript?

Sim, operadores relacionais são:

  • > siginifica maior que
  • < menor que
  • >= maior ou igual
  • < menor ou igual
  • == igual
  • === identico
  • != não igual(diferente)
  • !== desigual restrito
Operadores lógicos:
  • ! negação
  • && conjunção
  • || disconjunção

Sabe a diferença entre usar = ou == ou === em JavaScript?

Sim, o sinal de = significa recebe, == significa igual e === significa identico

dentro de uma expressão iremos fazer primeiro os operadores aritimeticos, depois vem os operadores relacionais, no qual não tem ordem de precedência, será feito o que vir primeiro da esquerda para a direita, e por ultimo os operadores lógicos na seguinte ordem ! && ||
